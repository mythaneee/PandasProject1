# Pokémon Data Analysis Project

This project demonstrates data analysis using pandas on the first generation Pokémon dataset.

## Project Structure

- `yourkerberos_pokemon.ipynb` — Jupyter notebook with all analysis code and visualizations.
- `FirstGenPokemon.csv` — Dataset containing information about the first generation Pokémon.

## Getting Started

1. **Requirements**
   - Python 3.7+
   - pandas
   - Jupyter Notebook or VS Code with Jupyter extension

2. **Setup**
   - Install dependencies:
     ```bash
     pip install pandas jupyter
     ```
   - Open `yourkerberos_pokemon.ipynb` in Jupyter or VS Code.

3. **Usage**
   - Run the notebook cells to explore and analyze the Pokémon data.
   - Modify or extend the analysis as needed.

## Dataset

- **Source:** `FirstGenPokemon.csv`
- **Description:** Contains stats and attributes for the original 151 Pokémon.
